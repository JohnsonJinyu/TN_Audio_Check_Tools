## ❓ 常见问题

### Q: 应用启动黑屏？
**A**: 这是正常的，需要等待 React 开发服务器启动（约 10-20 秒）

### Q: 如何修改窗口大小？
**A**: 编辑 `src/main/main.js` 中的 BrowserWindow 配置
```javascript
mainWindow = new BrowserWindow({
  width: 1400,    // 修改宽度
  height: 900,    // 修改高度
  // ...
});
```

### Q: 如何关闭开发者工具？
**A**: 在 `src/main/main.js` 注释掉：
```javascript
// if (isDev) {
//   mainWindow.webContents.openDevTools();
// }
```

### Q: 如何添加新页面？
**A**: 
1. 在 `src/renderer/pages/` 中创建新组件
2. 在 `App.jsx` 中导入
3. 在 `menuItems` 中添加菜单项
4. 在 `renderContent()` 中添加路由
